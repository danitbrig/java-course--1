מה זה תכנות מונחה עצמים (OOP)?

תכנות מונחה עצמים (Object Oriented Programming — OOP) זו דרך לכתוב תוכנות מחשב ע"י שימוש ב"עצמים" (אובייקטים). כל אובייקט הוא יחידה עצמאית שיש לה מאפיינים (תכונות) ופעולות (דברים שהיא יודעת לעשות).
💡 אנלוגיה מהחיים

חשוב על חשבון בנק כעל אובייקט. המאפיינים שלו: מספר חשבון, תאריך פתיחה, יתרה נוכחית. הפעולות שלו: הפקדה, משיכה, בירור יתרה. כל חשבון בנק הוא אובייקט נפרד — עם מספר חשבון משלו ויתרה משלו — אבל כולם "יודעים" לעשות את אותן פעולות.

📦

אובייקט (Object)

"דבר" בתוכנה — עם תכונות ופעולות. לדוגמה: חשבון בנק מסוים, לקוח מסוים.

📐

מחלקה (Class)

"תבנית" ליצירת אובייקטים. המחלקה מגדירה מה יהיו המאפיינים והפעולות — האובייקט הוא העותק הספציפי.

🧬

הורשה (Inheritance)

מחלקה יכולה "לרשת" ממחלקה אחרת — לקבל את כל התכונות שלה ולהוסיף עוד. חוסך שכפול קוד.

🔗

קשרים (Relations)

אובייקטים מתחברים זה לזה: בנק מכיל חשבונות, לכל חשבון יש בעלים, בכל חשבון מתבצעות תנועות.

📝 לזכור
מחלקה = תוכנית (Blueprint) ← אובייקט = מופע (Instance). המחלקה Account מגדירה מה זה חשבון, אבל חשבון ספציפי עם מספר 12345 ויתרה של 1,000 ₪ — זה האובייקט.
☕

שפת Java — הכרות ראשונה

Java היא אחת משפות התכנות הפופולריות בעולם, הושקה ב-1995, ועד היום משמשת לפיתוח אפליקציות Android, מערכות ארגוניות, ועוד. התכונה הייחודית שלה: "Write Once, Run Anywhere" — כותבים את הקוד פעם אחת והוא רץ על כל מחשב.
💡 למה Java?
כל שפות ה-OOP פועלות לפי אותם עקרונות. אחרי שתלמד Java, המעבר לשפות כמו C#, Python או JavaScript יהיה הרבה יותר קל — ההבדלים הם בעיקר בתחביר (syntax), לא ברעיונות.
🔒

Strongly Typed

כל משתנה חייב להיות מוגדר עם סוג (int, String...). השפה מגינה עליך משגיאות.

🔠

Case-Sensitive

counter ו-Counter הם שני דברים שונים לגמרי! שימו לב לאותיות גדולות וקטנות.

🏗️

מבנה תוכנית ב-Java

כל תוכנית Java בנויה ממחלקות (classes). בתוך המחלקה יש מתודות (methods = פונקציות). התוכנית מתחילה לרוץ תמיד ממתודה מיוחדת שנקראת main.
Java התוכנית הראשונה שלך 
public class MyFirstProgram {

    public static void main(String[] args) {
        System.out.println("!שלום עולם");
    }

}
בואו נפרק את זה שורה-שורה:
קוד מה זה אומר
public class MyFirstProgram מגדירים מחלקה בשם MyFirstProgram. הקובץ חייב להיקרא MyFirstProgram.java
{ } סוגריים מסולסלים מגדירים בלוק קוד — תחילה וסוף
public static void main(...) המתודה main — נקודת ההתחלה. כל תוכנית חייבת אחת כזו, תמיד באותו מבנה
System.out.println(...) פקודת הדפסה למסך — מה שבתוך הסוגריים יוצג למשתמש
; נקודה-פסיק בסוף כל פקודה — חובה ב-Java!
⚠️ שגיאות נפוצות של מתחילים
1. לשכוח נקודה-פסיק ; בסוף שורה → שגיאת קומפילציה
2. שם הקובץ לא תואם לשם המחלקה → השפה לא מוצאת את התוכנית
3. לכתוב Main במקום main (אות גדולה) → Java לא מזהה את נקודת ההתחלה
📝 הערות (Comments)
הערות הן הסברים שכותבים בתוך הקוד. הקומפיילר מתעלם מהן — הן לך ולצוות שלך:
// הערה עד סוף השורה
/* ... */ הערה מרובת שורות
📊

משתנים וסוגי נתונים

משתנה הוא "קופסה" בזיכרון המחשב שמחזיקה ערך. ב-Java חובה לומר מראש מה סוג הערך שיהיה בקופסה — מספר שלם? מספר עשרוני? טקסט?
💡 אנלוגיה

חשוב על משתנה כמו קופסה עם תווית. התווית אומרת מה מותר לשים בפנים: קופסה עם תווית "מספרים שלמים" (int) — אפשר לשים 5 או 100, אבל לא 3.14 ולא "שלום".

8 הסוגים הבסיסיים (Primitive Types):
סוג מה מאחסן דוגמה מתי נשתמש
int מספר שלם 42, -7, 0 ספירות, גילאים, כמויות
double מספר עשרוני 3.14, -0.5 ציונים, מחירים, חישובים
boolean אמת/שקר true, false כן/לא, פועל/כבוי
char תו בודד 'A', 'א', '7' ציון מילולי, סימן
String מחרוזת (טקסט) "Hello", "שלום" שמות, הודעות, כתובות
long מספר שלם גדול מאוד 9223372036854775807 מספרי ת.ז., זמנים
float עשרוני (פחות מדויק) 3.14f נדיר — בד"כ נעדיף double
byte / short מספרים שלמים קטנים 127 / 32767 חיסכון זיכרון (נדיר)
Java הגדרת משתנים 
// מספר שלם
int age = 25;

// מספר עשרוני
double price = 49.90;

// ערך בוליאני (אמת/שקר)
boolean isStudent = true;

// תו בודד — גרש בודד!
char grade = 'A';

// מחרוזת (טקסט) — גרשיים כפולים!
String name = "נעם";

// ניתן להגדיר מספר משתנים מאותו סוג
double width = 5.0, height = 3.5;
✅ מוסכמות שמות (Naming Conventions)
משתנים: מתחילים באות קטנה. אם יש כמה מילים → camelCase:  productPrice, isReady
מחלקות: מתחילות באות גדולה → PascalCase:  MyProgram, BankAccount
קבועים: אותיות גדולות עם קו תחתון:  MAX_VALUE, PI
Java קבועים (Constants) 
// קבוע = משתנה שלא ניתן לשנות לאחר שהוגדר
final double PI = 3.14159;
final int HOURS_PER_DAY = 24;

// PI = 2.71;  ← שגיאה! אסור לשנות קבוע
📺

פלט — הדפסה למסך

כדי להציג מידע למשתמש, משתמשים בפקודות ההדפסה של Java. יש שלוש גרסאות:
Java שלושת סוגי ההדפסה 
// print — מדפיס ונשאר באותה שורה
System.out.print("שלום ");
System.out.print("עולם");
// פלט: שלום עולם

// println — מדפיס ויורד שורה
System.out.println("שורה ראשונה");
System.out.println("שורה שנייה");
// פלט:
// שורה ראשונה
// שורה שנייה

// שרשור (חיבור) טקסט עם +
String name = "נעם";
int age = 25;
System.out.println("שמי " + name + " ואני בן/בת " + age);
Java הדפסה בתבנית (printf) 
String name = "דוד";
double avg = 92.6666;

// %s = מקום שמור למחרוזת, %n = ירידת שורה
System.out.printf("Student: %s%n", name);

// %.1f = מספר עשרוני עם ספרה אחת אחרי הנקודה
System.out.printf("Average: %.1f%n", avg);
// פלט: Average: 92.7
📝 תווים מיוחדים
\n — ירידת שורה  |  \t — Tab  |  \\ — להדפיס \  |  \" — להדפיס "
⌨️

קלט — קריאה מהמשתמש

כדי לקרוא נתונים מהמשתמש, משתמשים במחלקה Scanner. צריך לייבא אותה (import) בתחילת הקובץ.
Java קלט מהמשתמש — תוכנית מלאה 
import java.util.Scanner;   // חובה — מייבאים את Scanner

public class HelloUser {
    public static void main(String[] args) {

        // יוצרים אובייקט Scanner לקריאת קלט
        Scanner reader = new Scanner(System.in);

        // קולטים מחרוזת
        System.out.print("מה שמך? ");
        String name = reader.nextLine();

        // קולטים מספר שלם
        System.out.print("מה גילך? ");
        int age = reader.nextInt();

        // מציגים את הנתונים
        System.out.println("שלום " + name + "! אתה בן/בת " + age);

        reader.close();   // סוגרים את הסקנר
    }
}
פקודת קלט מה קוראת
reader.nextLine() שורת טקסט שלמה (String)
reader.nextInt() מספר שלם (int)
reader.nextDouble() מספר עשרוני (double)
reader.nextBoolean() true או false
⚠️ מלכודת נפוצה! nextInt + nextLine
אם קוראים nextInt() ומיד אחריו nextLine(), ה-nextLine "יבלע" את ה-Enter שנשאר מהקלט הקודם. הפתרון: להוסיף reader.nextLine(); ריק אחרי nextInt כדי "לנקות" את הזיכרון.
Java הפתרון למלכודת 
System.out.print("הכנס גיל: ");
int age = reader.nextInt();

reader.nextLine();   // ← ניקוי! בולע את ה-Enter

System.out.print("הכנס שם מלא: ");
String name = reader.nextLine();   // עכשיו ממתין לקלט כרגיל

תרגילים

הגיע הזמן לתרגל! נסה לכתוב כל תרגיל בעצמך לפני שאתה מציץ בפתרון.

✏️ תרגיל 1 — הצגת כרטיס ביקור

קל

כתוב תוכנית שמגדירה משתנים עבור: שם, גיל, עיר מגורים ומספר טלפון. התוכנית תדפיס למסך "כרטיס ביקור" בפורמט יפה, לדוגמה: 

===== כרטיס ביקור =====
שם: נעם כהן
גיל: 28
עיר: תל אביב
טלפון: 054-1234567
========================
Java פתרון תרגיל 1 
public class BusinessCard {
    public static void main(String[] args) {
        String name = "נעם כהן";
        int age = 28;
        String city = "תל אביב";
        String phone = "054-1234567";

        System.out.println("===== כרטיס ביקור =====");
        System.out.println("שם: " + name);
        System.out.println("גיל: " + age);
        System.out.println("עיר: " + city);
        System.out.println("טלפון: " + phone);
        System.out.println("========================");
    }
}

✏️ תרגיל 2 — מחשבון שטח מלבן

קל

כתוב תוכנית שקולטת מהמשתמש אורך ורוחב של מלבן (מספרים עשרוניים), מחשבת את השטח (אורך × רוחב) ואת ההיקף (2 × (אורך + רוחב)), ומדפיסה את התוצאות.

Java פתרון תרגיל 2 
import java.util.Scanner;

public class RectangleCalc {
    public static void main(String[] args) {
        Scanner reader = new Scanner(System.in);

        System.out.print("הכנס אורך: ");
        double length = reader.nextDouble();

        System.out.print("הכנס רוחב: ");
        double width = reader.nextDouble();

        double area = length * width;
        double perimeter = 2 * (length + width);

        System.out.println("שטח המלבן: " + area);
        System.out.println("היקף המלבן: " + perimeter);

        reader.close();
    }
}

✏️ תרגיל 3 — המרת טמפרטורה

בינוני

כתוב תוכנית שקולטת טמפרטורה בצלזיוס ומדפיסה אותה בפרנהייט.
הנוסחה: F = C × 9/5 + 32
שים לב: מה קורה כשמחלקים שני שלמים ב-Java? (רמז: 9/5 ≠ 1.8)

Java פתרון תרגיל 3 
import java.util.Scanner;

public class TempConverter {
    public static void main(String[] args) {
        Scanner reader = new Scanner(System.in);

        System.out.print("הכנס טמפרטורה בצלזיוס: ");
        double celsius = reader.nextDouble();

        // שים לב: 9.0 ולא 9 — כדי לקבל חילוק עשרוני!
        double fahrenheit = celsius * 9.0 / 5 + 32;

        System.out.printf("%.1f°C = %.1f°F%n", celsius, fahrenheit);

        reader.close();
    }
}
💡 נקודת מפתח
9 / 5 = 1 (חילוק שלמים!) אבל 9.0 / 5 = 1.8 (חילוק עשרוני). מספיק שאחד מהמספרים יהיה עשרוני כדי שהתוצאה תהיה עשרונית.

מה ההבדל בין print ל-println?

אין הבדל

📋 סיכום שיעור 1 — מה למדנו?

  • מה זה OOP — אובייקטים, מחלקות, מאפיינים ופעולות
  • Java — שפה מונחית עצמים, Case-Sensitive, Strongly Typed
  • מבנה תוכנית: class → main method → פקודות
  • סוגי נתונים: int, double, boolean, char, String
  • הגדרת משתנים וקבועים (final)
  • פלט: print, println, printf
  • קלט: Scanner — nextLine, nextInt, nextDouble
  • מלכודת nextInt + nextLine ואיך לפתור
  • חילוק שלמים (5/2 = 2) לעומת חילוק עשרוני
🔜 בשיעור הבא
ביטויים חישוביים, תנאים (if/else), ולולאות (for, while) — נלמד איך לגרום לתוכנית לקבל החלטות ולחזור על פעולות.
