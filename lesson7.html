בנאי (Constructor)

בנאי הוא מתודה מיוחדת שנקראת אוטומטית כשיוצרים אובייקט עם new. שמו חייב להיות זהה לשם המחלקה ואין לו סוג החזרה.
JavaConstructor
public class Student {
    String name;
    int age;

    // בנאי — שם זהה למחלקה, בלי סוג החזרה
    public Student(String name, int age) {
        this.name = name;
        this.age = age;
    }
}

// שימוש — נקי ופשוט!
Student s = new Student("נעם", 25);
📝 בנאי ברירת מחדל
אם לא כתבת שום בנאי, Java מספקת בנאי ריק אוטומטית. אבל — ברגע שכתבת בנאי אחד, הבנאי האוטומטי נעלם!
👆

this — הפניה לאובייקט הנוכחי

this פירושו "האובייקט הנוכחי". משתמשים בו כשלפרמטר יש אותו שם כמו לשדה — כדי להבדיל ביניהם.
Javathis
public Student(String name, int age) {
    this.name = name;  // this.name = שדה, name = פרמטר
    this.age = age;    // this.age = שדה, age = פרמטר
}

Encapsulation — הסתרת נתונים

Encapsulation (כימוס) = הגנה על השדות מגישה ישירה מבחוץ. מסמנים שדות כ-private ונותנים גישה רק דרך מתודות מבוקרות.
🔓

public

כולם יכולים לגשת — מתוך המחלקה ומבחוץ.

🔒

private

רק מתוך המחלקה עצמה — אי אפשר לגשת מבחוץ.

Javaprivate + Getters/Setters
public class Student {
    private String name;
    private int age;

    public Student(String name, int age) {
        this.name = name;
        setAge(age);  // שימוש ב-setter גם בבנאי — ולידציה!
    }

    // Getter — קריאה
    public String getName() { return name; }
    public int getAge() { return age; }

    // Setter — כתיבה עם ולידציה
    public void setAge(int age) {
        if (age > 0 && age < 120) {
            this.age = age;
        }
    }
}
✅ למה Encapsulation?
1. ולידציה: ה-setter יכול לבדוק תקינות (גיל חיובי)
2. גמישות: אפשר לשנות מימוש פנימי בלי לשבור קוד חיצוני
3. אבטחה: מונע שינוי מקרי/זדוני של נתונים

toString() — ייצוג טקסטי

JavatoString
public class Student {
    private String name;
    private int age;

    // toString — נקראת אוטומטית ב-println!
    public String toString() {
        return "Student{name='" + name + "', age=" + age + "}";
    }
}

// שימוש
Student s = new Student("נעם", 25);
System.out.println(s);  // Student{name='נעם', age=25}

📋 סיכום שיעור 7

  • Constructor = מתודה מיוחדת ליצירת אובייקטים, שם זהה למחלקה
  • this = הפניה לאובייקט הנוכחי, מבדיל שדה מפרמטר
  • private = הסתרה, public = חשיפה
  • Getters = קריאת שדות, Setters = עדכון עם ולידציה
  • toString() = ייצוג טקסטי, נקרא אוטומטית ב-println
  • Encapsulation = עיקרון מפתח ב-OOP
