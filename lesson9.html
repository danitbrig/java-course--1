מהו פולימורפיזם?

פולימורפיזם (Polymorphism = "ריבוי צורות") — היכולת לפנות לאובייקטים מסוגים שונים דרך אותו טיפוס בסיס, וכל אובייקט יגיב לפי המחלקה האמיתית שלו.
Javaפולימורפיזם בפעולה
Animal[] zoo = {
    new Dog("Rex", 3, "Lab"),
    new Cat("Whiskers", 5),
    new Bird("Tweety", 2)
};

// כל אחד מגיב אחרת!
for (Animal a : zoo) {
    a.speak();
}
// Rex barks! Woof!
// Whiskers says Meow!
// Tweety sings Tweet tweet!
📝 Dynamic Binding
Java מחליטה בזמן ריצה (לא בקומפילציה) איזו גרסה של speak() להריץ — לפי הטיפוס האמיתי של האובייקט.

Casting ו-instanceof

Javaבדיקת סוג אובייקט
Animal a = new Dog("Rex", 3, "Lab");

// instanceof בודק את הסוג האמיתי
if (a instanceof Dog) {
    Dog d = (Dog) a;  // Downcasting
    d.fetch();         // עכשיו אפשר לקרוא למתודות של Dog
}

Abstract Classes — מחלקות מופשטות

מחלקה אבסטרקטית היא מחלקה שאי אפשר ליצור ממנה אובייקטים ישירות — היא משמשת רק כבסיס להורשה.
JavaAbstract Class
public abstract class Shape {
    protected String color;

    public Shape(String color) {
        this.color = color;
    }

    // מתודה אבסטרקטית — בלי גוף! מחלקות בנות חייבות לממש
    public abstract double getArea();

    // מתודה רגילה — כל הבנות מקבלות אותה
    public void printInfo() {
        System.out.println(color + " shape, area: " + getArea());
    }
}

public class Circle extends Shape {
    private double radius;

    public Circle(String color, double radius) {
        super(color);
        this.radius = radius;
    }

    // חייבים לממש!
    @Override
    public double getArea() {
        return Math.PI * radius * radius;
    }
}

// Shape s = new Shape("red");  ← שגיאה! אי אפשר ליצור אובייקט מ-abstract
Shape s = new Circle("red", 5);  // ✅ זה עובד!
s.printInfo();  // red shape, area: 78.54

📋 סיכום שיעור 9

  • פולימורפיזם = אותו טיפוס, התנהגויות שונות בזמן ריצה
  • Dynamic Binding = Java בוחרת מתודה לפי הסוג האמיתי
  • instanceof = בדיקת סוג אובייקט
  • abstract class = אי אפשר ליצור ממנה אובייקט ישירות
  • abstract method = בלי גוף, מחלקות בנות חייבות לממש
  • מחלקה עם מתודה אבסטרקטית חייבת להיות abstract
