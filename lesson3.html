מחרוזות (Strings) — יצירה ופעולות

String ב-Java הוא אובייקט (לא טיפוס פרימיטיבי), ולכן מתחיל באות גדולה. מחרוזת היא בלתי ניתנת לשינוי (Immutable) — כל פעולה מחזירה מחרוזת חדשה.
Javaיצירת מחרוזות
// דרך מקוצרת (הנפוצה)
String name = "Java Programming";

// דרך מלאה (פחות נפוצה)
String name2 = new String("Java Programming");

// שרשור מחרוזות עם +
String full = "Hello" + " " + "World";  // "Hello World"

// שרשור עם מספרים
int age = 25;
String msg = "גיל: " + age;  // "גיל: 25"

📝 Immutable = בלתי ניתן לשינוי

text.toUpperCase() לא משנה את text — צריך לשמור: text = text.toUpperCase()

השוואת מחרוזות — equals ולא ==

זה אחד הנושאים הכי חשובים בשיעור הזה. ב-Java, == על מחרוזות בודק אם זה אותו אובייקט בזיכרון, לא אם התוכן זהה!
⚠️ כלל ברזל: להשוות מחרוזות תמיד עם equals()
== → בודק אם זה אותו אובייקט (כתובת בזיכרון)
equals() → בודק אם התוכן זהה ✅
equalsIgnoreCase() → בודק תוכן, מתעלם מגודל אותיות ✅
Java== לעומת equals()
String a = "Java";
String b = new String("Java");

System.out.println(a == b);         // false! (אובייקטים שונים)
System.out.println(a.equals(b));    // true  (תוכן זהה) ✅

// התעלמות מגודל אותיות
String c = "JAVA";
System.out.println(a.equalsIgnoreCase(c));  // true ✅
💡 אנלוגיה

חשוב על שני עותקים של אותו ספר. == שואל "האם זה אותו עותק פיזי?" (לא). equals שואל "האם התוכן זהה?" (כן).

JavacompareTo — השוואה לקסיקוגרפית
String a = "Apple";
String b = "Banana";

int result = a.compareTo(b);
// result < 0 → a בא לפני b (לפי ABC)
// result == 0 → שווים
// result > 0 → a בא אחרי b

System.out.println(result);  // מספר שלילי (A לפני B)

✏️ תרגיל 1 — ספירת תווים

קל

כתוב תוכנית שקולטת מחרוזת ומדפיסה את אורכה, האות הראשונה והאחרונה.

Javaפתרון
import java.util.Scanner;

public class CharCount {
    public static void main(String[] args) {
        Scanner reader = new Scanner(System.in);
        System.out.print("הכנס מחרוזת: ");
        String text = reader.nextLine();

        System.out.println("אורך: " + text.length());
        System.out.println("אות ראשונה: " + text.charAt(0));
        System.out.println("אות אחרונה: " + text.charAt(text.length() - 1));
        reader.close();
    }
}

✏️ תרגיל 2 — בדיקת פלינדרום

בינוני

כתוב תוכנית שקולטת מחרוזת ובודקת אם היא פלינדרום (נקראת אותו דבר מההתחלה ומהסוף). לדוגמה: "aba", "racecar".

Javaפתרון
import java.util.Scanner;

public class Palindrome {
    public static void main(String[] args) {
        Scanner reader = new Scanner(System.in);
        System.out.print("הכנס מילה: ");
        String word = reader.nextLine().toLowerCase();

        boolean isPalindrome = true;
        for (int i = 0; i < word.length() / 2; i++) {
            if (word.charAt(i) != word.charAt(word.length() - 1 - i)) {
                isPalindrome = false;
                break;
            }
        }

        System.out.println(isPalindrome ? "פלינדרום!" : "לא פלינדרום");
        reader.close();
    }
}

✏️ תרגיל 3 — מחשבון שטח עיגול

קל

קלוט רדיוס עיגול וחשב את השטח (πr²) ואת ההיקף (2πr) באמצעות Math.PI.

Javaפתרון
import java.util.Scanner;

public class CircleCalc {
    public static void main(String[] args) {
        Scanner reader = new Scanner(System.in);
        System.out.print("הכנס רדיוס: ");
        double r = reader.nextDouble();

        double area = Math.PI * Math.pow(r, 2);
        double circumference = 2 * Math.PI * r;

        System.out.printf("שטח: %.2f%n", area);
        System.out.printf("היקף: %.2f%n", circumference);
        reader.close();
    }
}

✏️ תרגיל 4 — חיפוש case-insensitive

בינוני

קלוט מערך של מחרוזות ומחרוזת חיפוש. הדפס את כל האינדקסים שבהם נמצאת המחרוזת (בלי רגישות לאותיות).

Javaפתרון
public class SearchArray {
    public static void main(String[] args) {
        String[] names = {"Java", "python", "JAVA", "C++", "java"};
        String search = "java";

        boolean found = false;
        for (int i = 0; i < names.length; i++) {
            if (names[i].equalsIgnoreCase(search)) {
                System.out.println("נמצא באינדקס " + i);
                found = true;
            }
        }
        if (!found) {
            System.out.println("לא נמצא");
        }
    }
}
