מהי מחלקה (Class)?

מחלקה היא תבנית ליצירת אובייקטים. היא מגדירה אילו שדות (נתונים) ואילו מתודות (פעולות) יהיו לכל אובייקט שנוצר ממנה.
💡 אנלוגיה

מחלקה = תוכנית אדריכלית של בית. אובייקט = בית שנבנה לפי התוכנית. מתוכנית אחת אפשר לבנות הרבה בתים — כל אחד עם צבע שונה, אבל אותו מבנה.

Javaמחלקה בסיסית — Student
public class Student {
    // שדות (Fields) — הנתונים של כל סטודנט
    String name;
    int age;
    double grade;

    // מתודה — פעולה שסטודנט יודע לעשות
    void introduce() {
        System.out.println("שמי " + name + ", גיל " + age);
    }

    // מתודה שמחזירה ערך
    boolean isPassing() {
        return grade >= 60;
    }
}
📋

שדות (Fields) — הנתונים

שדות הם משתנים ששייכים לאובייקט. כל אובייקט מחזיק עותק משלו של השדות — לסטודנט אחד שם "נעם" ולשני "דנית".
📝 שדות לעומת משתנים מקומיים
שדה: מוגדר בתוך המחלקה (מחוץ למתודות), חי כל עוד האובייקט חי.
משתנה מקומי: מוגדר בתוך מתודה, חי רק בזמן ריצת המתודה.
⚡

מתודות — הפעולות

מתודות של אובייקט לא צריכות static! הן שייכות לאובייקט ויכולות לגשת ישירות לשדות שלו.
Javaמתודות אובייקט
public class BankAccount {
    String owner;
    double balance;

    void deposit(double amount) {
        balance += amount;
        System.out.println("הופקד " + amount + ". יתרה: " + balance);
    }

    void withdraw(double amount) {
        if (amount <= balance) {
            balance -= amount;
            System.out.println("נמשך " + amount + ". יתרה: " + balance);
        } else {
            System.out.println("אין מספיק כסף!");
        }
    }
}
✨

יצירת אובייקטים — new

Javaיצירה ושימוש באובייקטים
public class Main {
    public static void main(String[] args) {
        // יצירת אובייקט חדש
        Student s1 = new Student();
        s1.name = "נעם";
        s1.age = 25;
        s1.grade = 88.5;

        // שימוש במתודות
        s1.introduce();    // שמי נעם, גיל 25
        System.out.println(s1.isPassing());  // true

        // אובייקט שני — נתונים שונים, אותו מבנה
        Student s2 = new Student();
        s2.name = "דנית";
        s2.age = 23;
        s2.grade = 95.0;
    }
}
⚠️ new חובה!
בלי new, אין אובייקט! Student s1; מגדיר רק משתנה — הוא מכיל null. ניסיון לגשת לשדות יגרום ל-NullPointerException.

תרגילים

✏️ תרגיל — בניית מחלקת Rectangle

בינוני

צור מחלקה Rectangle עם שדות width ו-height, מתודות לחישוב שטח והיקף, ומתודה isSquare() שבודקת אם זה ריבוע.

Java
public class Rectangle {
    double width;
    double height;

    double getArea() {
        return width * height;
    }

    double getPerimeter() {
        return 2 * (width + height);
    }

    boolean isSquare() {
        return width == height;
    }
}

📋 סיכום שיעור 6

  • Class = תבנית, Object = מופע (instance) של התבנית
  • שדות (Fields) = נתונים, מתודות (Methods) = פעולות
  • new יוצר אובייקט חדש בזיכרון
  • מתודות אובייקט לא צריכות static
  • כל אובייקט מחזיק עותק עצמאי של השדות
  • null = אובייקט לא נוצר, NullPointerException = ניסיון גישה ל-null
