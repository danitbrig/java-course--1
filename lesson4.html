מערכים (Arrays) — מבוא

מערך הוא מבנה נתונים שמאחסן כמה ערכים מאותו סוג ברצף. בשונה מרשימה ב-Python, מערך ב-Java הוא בגודל קבוע — לא ניתן להגדיל או להקטין אותו.
💡 אנלוגיה

חשוב על מערך כמו ארון עם מגירות ממוספרות. כשקונים ארון עם 5 מגירות — אי אפשר להוסיף מגירה שישית. אבל אפשר לשים ולהוציא דברים מכל מגירה לפי המספר שלה (0, 1, 2, 3, 4).

Javaיצירת מערכים
// דרך 1: הגדרה עם ערכים
int[] numbers = {10, 20, 30, 40, 50};

// דרך 2: הגדרה עם גודל (ערכים ריקים = 0)
int[] scores = new int[5];  // [0, 0, 0, 0, 0]

// דרך 3: גודל דינמי מהמשתמש
int size = reader.nextInt();
int[] arr = new int[size];

// מערכים מסוגים שונים
double[] prices = new double[10];
String[] names = {"Alice", "Bob", "Charlie"};
boolean[] flags = new boolean[3];  // [false, false, false]
⚠️ ערכי ברירת מחדל במערך ריק
int[] → 0  |  double[] → 0.0  |  boolean[] → false  |  String[] → null
🔍

גישה לאיברים ו-length

Javaגישה, עדכון ומעבר על מערך
int[] nums = {10, 20, 30, 40, 50};

// גישה לאיבר (אינדקס מתחיל מ-0!)
System.out.println(nums[0]);   // 10
System.out.println(nums[4]);   // 50

// עדכון ערך
nums[2] = 99;  // [10, 20, 99, 40, 50]

// אורך המערך — .length בלי סוגריים!
System.out.println(nums.length);  // 5

// מעבר עם for
for (int i = 0; i < nums.length; i++) {
    System.out.println("אינדקס " + i + ": " + nums[i]);
}

// מעבר עם for-each (קריאה בלבד!)
for (int num : nums) {
    System.out.println(num);
}
⚠️ ArrayIndexOutOfBoundsException
גישה לאינדקס שלא קיים (למשל nums[5] במערך בגודל 5) תגרום לשגיאה! האינדקס האחרון הוא תמיד length - 1.
📝 .length במערך לעומת .length() במחרוזת
arr.length — בלי סוגריים (תכונה של מערך)
str.length() — עם סוגריים (מתודה של String)
זה הבדל קלאסי ששואלים עליו במבחן!

אלגוריתמים על מערכים

Javaחיפוש ערך מינימלי
public static int findMin(int[] arr) {
    int min = arr[0];
    for (int i = 1; i < arr.length; i++) {
        if (arr[i] < min) {
            min = arr[i];
        }
    }
    return min;
}
Javaחיפוש סדרתי (Sequential Search)
public static int search(int[] arr, int target) {
    for (int i = 0; i < arr.length; i++) {
        if (arr[i] == target) {
            return i;  // נמצא — מחזיר אינדקס
        }
    }
    return -1;  // לא נמצא
}
Javaחישוב ממוצע
public static double average(int[] arr) {
    int sum = 0;
    for (int num : arr) {
        sum += num;
    }
    return (double) sum / arr.length;
}

מערכים דו-ממדיים (2D Arrays)

מערך דו-ממדי הוא מערך של מערכים — כמו טבלה עם שורות ועמודות.
Javaמערך דו-ממדי
// 3 שורות × 4 עמודות
int[][] matrix = new int[3][4];

// עם ערכים
int[][] grid = {
    {1, 2, 3},
    {4, 5, 6},
    {7, 8, 9}
};

// גישה: grid[שורה][עמודה]
System.out.println(grid[1][2]);  // 6

// מעבר על כל האיברים
for (int i = 0; i < grid.length; i++) {        // שורות
    for (int j = 0; j < grid[i].length; j++) {  // עמודות
        System.out.print(grid[i][j] + " ");
    }
    System.out.println();  // ירידת שורה
}

// grid.length = מספר שורות (3)
// grid[0].length = מספר עמודות בשורה 0 (3)

תרגילים

✏️ תרגיל 1 — ספירת חיוביים

קל

כתוב פונקציה שמקבלת מערך של int ומחזירה את מספר הערכים החיוביים.

Javaפתרון
public static int countPositive(int[] arr) {
    int count = 0;
    for (int num : arr) {
        if (num > 0) {
            count++;
        }
    }
    return count;
}

✏️ תרגיל 2 — בדיקת סדרה עולה

בינוני

כתוב פונקציה שמקבלת מערך ומחזירה true אם כל איבר גדול מהקודם (סדרה עולה).

Javaפתרון
public static boolean isAscending(int[] arr) {
    for (int i = 1; i < arr.length; i++) {
        if (arr[i] <= arr[i - 1]) {
            return false;
        }
    }
    return true;
}

int[][] table = new int[10][10];

for (int i = 0; i < 10; i++) {

מה ההבדל בין .length ל-.length()?

.length = מערך (תכונה), .length() = מחרוזת (מתודה)
אין הבדל
.length() = מערך, .length = מחרוזת

📋 סיכום שיעור 4

  • מערך = גודל קבוע, סוג אחיד, אינדקס מ-0
  • הגדרה: int[] arr = {1,2,3} או new int[size]
  • .length בלי סוגריים (תכונה, לא מתודה)
  • ערכי ברירת מחדל: int→0, boolean→false, String→null
  • for-each לקריאה בלבד: for (int x : arr)
  • אלגוריתמים: חיפוש סדרתי, מינימום, ממוצע
  • מערך 2D: int[][] matrix, גישה: matrix[שורה][עמודה]
  • ArrayIndexOutOfBoundsException — אינדקס מחוץ לטווח
