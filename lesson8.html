מהי הורשה?

הורשה מאפשרת ליצור מחלקה חדשה שמבוססת על מחלקה קיימת. המחלקה החדשה (בת) מקבלת את כל השדות והמתודות של מחלקת האב, ויכולה להוסיף או לשנות.
💡 אנלוגיה

מחלקת אב: רכב (יש לו צבע, מהירות, יכול לנסוע ולעצור). מחלקת בת: מכונית חשמלית — יש לה הכל של רכב + רמת סוללה ויכולת טעינה.

🔗

extends — ירושה

Javaהורשה בסיסית
// מחלקת אב
public class Animal {
    protected String name;
    protected int age;

    public Animal(String name, int age) {
        this.name = name;
        this.age = age;
    }

    public void speak() {
        System.out.println(name + " makes a sound");
    }
}

// מחלקת בת — יורשת מ-Animal
public class Dog extends Animal {
    private String breed;

    public Dog(String name, int age, String breed) {
        super(name, age);  // קריאה לבנאי של האב
        this.breed = breed;
    }

    // Override — דריסת מתודה של האב
    @Override
    public void speak() {
        System.out.println(name + " barks! Woof!");
    }

    // מתודה חדשה — רק ל-Dog
    public void fetch() {
        System.out.println(name + " fetches the ball!");
    }
}
⬆️

super — הפניה למחלקת האב

super() קורא לבנאי של האב. חייב להיות בשורה הראשונה של בנאי הבת!
super.methodName() קורא למתודה של האב (גם אם דרסנו אותה).
📝 protected
protected = גלוי למחלקה עצמה ולמחלקות שיורשות ממנה. ביניים בין private ל-public.
🔄

Override — דריסת מתודות

Javaשימוש
Animal a = new Animal("Generic", 5);
Dog d = new Dog("Rex", 3, "Labrador");

a.speak();  // Generic makes a sound
d.speak();  // Rex barks! Woof! (דרוסה!)
d.fetch();  // Rex fetches the ball!
✅ @Override
תמיד שימו @Override מעל מתודה דרוסה. זה לא חובה, אבל מגן מטעויות — אם טעיתם בשם, הקומפיילר יתריע.

📋 סיכום שיעור 8

  • extends = מחלקת בת יורשת ממחלקת אב
  • super() = קריאה לבנאי האב (חייב בשורה הראשונה)
  • @Override = דריסת מתודה של האב במחלקת הבת
  • protected = גלוי למחלקה ולבנות שלה
  • Java תומכת רק בהורשה בודדת (אב אחד)
  • כל מחלקה ב-Java יורשת מ-Object
